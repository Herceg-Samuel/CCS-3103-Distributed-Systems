# Distributed Event Ordering Simulator

This mini project models a small distributed system with three processes, internal events, message passing, Lamport clock updates, and event logging.

## Files
- lamport_clock.py: Implements the Lamport clock logic.
- simulator.py: Runs the simulation and writes the deliverables.
- event_log.txt: Generated event log.
- timeline_diagram.txt: Generated timeline visualization.
- performance_analysis.txt: Generated performance notes.
- reflection.txt: Reflection on Lamport clock limitations.

## Run
```bash
python3 simulator.py
```
